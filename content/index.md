# Knowledge Garden

A structured reference on world religions, spiritual traditions, and esoteric history — built for study and research.

## What's Here

This collection covers 139 notes across six regional and thematic sections:

| Section | Focus |
|---|---|
| **Africa** | Ancient Egyptian religion, Yoruba, Kongo, indigenous African traditions |
| **Americas** | Mesoamerican religions, Afro-diasporic traditions (Vodou, Candomblé, Santería) |
| **Asia** | Hinduism, Buddhism, Daoism, Confucianism, Jainism, Shinto, Sikhism, Tibetan Buddhism |
| **Europe** | Norse, Greek, Roman, Celtic, Slavic paganism; Western esotericism, Gnosticism, occult revival |
| **Middle East** | Judaism, Christianity, Islam, Zoroastrianism, Sufism, Kabbalah, Gnosticism |
| **Cross-Regional** | Comparative themes: afterlife models, sacrifice, mysticism, sacred law, syncretism |

## How to Navigate

- Use the **search bar** (top left) to find any topic
- Use the **explorer** (left sidebar) to browse by region
- Use the **graph view** (right sidebar) to explore connections between traditions
- Each note links to related topics — follow the thread

## Standards

Notes are written to research standards: historically grounded, source-aware, and explicit about where traditions are commonly misrepresented. Where the scholarship is contested, that's noted.

This is a living collection. Notes are updated as new research is added.
